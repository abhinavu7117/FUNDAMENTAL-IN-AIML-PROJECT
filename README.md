# 📊 Smart Study Dashboard

A simple yet powerful productivity web app that helps students manage tasks, track progress, and stay consistent with their studies.

---

## 🚀 Live Demo

👉 https://abhinavu7117.github.io/FUNDAMENTAL-IN-AIML-PROJECT/

---

## 📌 Problem
As a student, managing assignments, deadlines, and study tasks can get overwhelming. Most of the time, tasks are forgotten or completed at the last minute, leading to stress and poor time management.

---

## 💡 Solution
This project provides a clean and interactive dashboard where users can:
- Add and manage tasks
- Track completion progress
- View statistics visually
- Stay aware of deadlines

---

## ✨ Features
- ✅ Add, edit, and delete tasks  
- 📅 Deadline tracking  
- ⚠️ Urgent task highlighting  
- 🗂️ Category-based organization  
- 📊 Graph (Completed vs Pending tasks)  
- 📈 Progress bar  
- 💾 Data saved using localStorage  

---

## 🛠️ Tech Stack
- HTML  
- CSS (Glassmorphism UI)  
- JavaScript  
- Chart.js
